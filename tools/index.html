Tools

Interactive tools built around the decisions OT security practitioners actually face. No signup required except for the RFP Evaluation Kit.

  • Evaluation
    Vendor comparison tool
    Filter the OT security market by protocol coverage, deployment model, industry, compliance framework, and company size. Covers established and challenger vendors across all categories.
    ›
  • Evaluation
    OT Security RFP Evaluation Kit
    Four fillable PDF documents covering the full evaluation arc: vendor briefing template, PoC success criteria, protocol and asset questionnaire, and evaluation decision tracker. Email required for download.
    ›
  • Compliance
    Asset Scoping Decision Tool
    Walk through the classification decision for individual OT assets under NERC CIP, NIS2, or both. Produces a documented classification with the criteria applied — suitable for audit records.
    ›
  • Architecture
    Firewall vs. Data Diode TCO Calculator
    Enter your environment parameters and find the exact month where the data diode's higher upfront cost is recovered by firewall OpEx savings. Produces a CFO-ready output for budget conversations.
    ›
  • Operations
    OT SOC Integration Health Scorecard
    Benchmark your OT SOC integration across six key metrics — detection coverage, false positive rate, mean time to detect, triage time, escalation accuracy, and maintenance window alignment. Identifies what to fix first.
